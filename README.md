## Práctica Formativa Obligatoria 1

## Descripción del Proyecto

Este proyecto corresponde a un Trabajo Práctico de Front End. El objetivo es diseñar y desarrollar una página web utilizando HTML y CSS, aplicando conceptos fundamentales como estructura semántica, estilos personalizados, uso de Google Fonts, diseño responsivo y organización del contenido.

La página incluye distintas secciones como presentación personal, tarjetas de proyectos, listado de habilidades, formulario de contacto y películas favoritas, cumpliendo con todos los requerimientos solicitados.
